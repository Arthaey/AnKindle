MDX-Kindle

Features:

  • One-Click-Import Kindle vocabulary
  • Provide a default template for words（also can be customized）
  • Query MDX dicts for words
  • Switch MDX file for different language keyword

Optional:MDX dictionary is optional, you dan use your won MDX files for query.

Prerequisites

  1. Ensure your Kindle has been adapted to machine.
  2. In Anki select: Tools > "MDX-Kindle"

使用步骤

请依次点击【导入】窗体内的按钮：

  1. （必选）保存为笔记类型

    选择Kindle单词导出的目标笔记类型，本插件默认提供一个类型为【AnkiMDXKindle】， 您可以在Anki菜单中的【工具】>【笔记类型管理】中查看其具体内容。

    自定义模板：除了默认的模板【AnkiMDXKindle】，您可以自定义自己的模板进行选择，但是在您的模板中 必须实现以下全部字段：（顺序可不同，允许有其他字段，大小写敏感）

    stem
    单词原型
    word
    单词
    lang
    源语言类型
    creation_tm
    加入单词本的时间
    usage
    Kindle引用的原句
    title
    书名
    authors
    作者
    id
    ID
    mdx_dict
    用户储存本插件的MDX字典的查询内容

    以上字段必须全部实现，插件才能识别并显示在笔记类型选项中。

  2. （必选）保存到记忆库

    选择Kindle单词的目标保存记忆库。

  3. （可选）MDX词典

    模板提供一个字段为【mdx_dict】您可以选择自己喜欢的MDX字典，插件如果此选项被提供，那么插件会查询这个 词典然后把可用的查询内容放进【mdx_dict】

  4. 导入

    执行导入
